Project Summary

The purpose of this project is to determine if there is a relationship between certain weather patterns and latitude. Weather data for over 500 randomly selected cities was used for this analysis.

The weather data - max temperature, humidity, wind speed, and cloudiness - was imported using the OpenWeatherMap API, then plotted into scatterplots using Matplotlib for visualization.
