Summary: Latitude vs. X

The purpose of this project is to analyze how weather changes closer to the equator. The weather data was first pulled from the OpenWeatherMap API to compile a dataset including over 500 randomly selected cities.

After assembling this dataset, we used Matplotlib to plot various aspects of the weather versus latitude. We specifically looked at maximum temperature(F), humidity(%), cloudiness(%), and windspeed(mph). The dataset, visualizations, and explanations of trends and correlation are provided on this site.
